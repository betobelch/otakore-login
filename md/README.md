🚀 Otakore - Tela de Login / Cadastro

Interface moderna de autenticação (Login/Sign Up) desenvolvida com foco em design elegante, experiência do usuário e responsividade. O projeto apresenta um layout dividido com formulário interativo e uma seção visual com imagem temática.

Tela com:

Formulário de cadastro

Layout moderno com gradientes
Design responsivo
🧠 Sobre o Projeto

O Otakore é uma interface de autenticação front-end criada utilizando HTML, CSS e JavaScript puro.
Ideal para estudos, portfólio ou como base para integração com back-end (Firebase, Node.js, etc).

🛠️ Tecnologias Utilizadas
HTML5
CSS3 (Flexbox + Responsividade)
JavaScript (Vanilla JS)
Google Fonts
✨ Funcionalidades
✅ Validação de formulário
✅ Verificação de senha (mínimo 6 caracteres)
✅ Confirmação de senha
✅ Feedback visual com alertas
✅ Botão de login com Google (simulado)
✅ Efeitos visuais ao focar nos inputs
✅ Layout totalmente responsivo
